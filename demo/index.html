Get Started

This page is an overview of @frontend-trends/web-components.

@frontend-trends/web-components is a open source chart library with VanillaJS Web Components that follow the open-wc recommendation.

Installation

$ npm i @frontend-trends/web-components --save

Not all browsers support the Web Components specs. Therefore it is better to load the following polyfill on the page. 

<script src="./node_modules/@webcomponents/webcomponentsjs/webcomponents-bundle.js"></script>

Donut Chart

<script type="module">
    import './node_modules/@frontend-trends/web-components/donut-chart.js';
</script>

<h2>Favorite cars</h2>

<donut-chart 
    width="250"
    radius="40"
    stroke-width="20"
    legend="right"
    source="/demo/api/cars.json"
></donut-chart>

Cars of my friends

Attributes

Attribute Default Description
width 250 chart width
height 250 chart height
legend right legend position
highlight #a8d1ff highlight color
radius 40 donut radius
stroke-width 30 donut stroke width
source - URL
